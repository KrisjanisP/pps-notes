## "teikums2" līdz 16:25

uzdevums:

> Ievaddatos vienīgajā rinda ir teikums, kurš vienmēr beidzas ar punktu. Jānosaka šī teikuma garākā vārda burtu skaits.

risinājums:

```cpp
#include <iostream>
using namespace std;
int main()
{
    string line;

    getline(cin, line);

    int res = 0, cur = 0;
    for(int i=0;i<line.size();i++){
        if(line[i]==' '||line[i]=='.')
            cur = 0;
        else {
            cur++;
            if(cur>res)
                res = cur;
        }
    }
    cout<<res<<endl;
}
```
