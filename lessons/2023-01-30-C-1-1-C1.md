## masīvu atkārtojums:

uzdevums:

> Pēc piecu skaitļu ievadīšanas izvadīt tos python saraksta pierakstā.

ievaddati:

```
1 2 3 4 5
```

izvaddati:

```
[1,2,3,4,5]
```

risinājums:

```cpp
#include <iostream>
using namespace std;
int main() {
    int mas[5];
    for(int i=0;i<5;i++)
        cin>>mas[i];
    cout<<'[';
    for(int i=0;i<5;i++){
        cout<<mas[i];
        if(i<4)
            cout<<',';
    }
    cout<<']';
}
```

## vector atkārtojums:

uzdevums:

> Pēc skaitļu virknes ievadīšanas, kas beidzās ar 0, izvadīt šo virkni python pierakstā.

ievaddati:

```
1 2 3 4 5 6 0
```

izvaddati:

```
[1,2,3,4,5,6]
```

risinājums:

```cpp
#include <iostream>
#include <vector>
using namespace std;
int main() {
    vector<int> vektors;

    int x;
    cin>>x;
    for(;x!=0;cin>>x){
        vektors.push_back(x);
    }
    cout<<'[';
    for(int i=0;i<vektors.size();i+=1) {
        cout<<vektors[i];
        if(i<vektors.size()-1)
            cout<<',';
    }
    cout<<']';
}
```
